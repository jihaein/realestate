# 부동산 매물 리스트 뷰어 (Windows)

## 설치 방법

1. ZIP 파일의 압축을 해제합니다.
2. 압축 해제된 폴더에서 `부동산매물뷰어.exe`를 실행합니다.
   - Windows 보안 경고가 표시되면 "추가 정보" → "실행"을 클릭합니다.

## 사용 방법

1. 프로그램이 시작되면 자동으로 최신 매물 데이터를 불러옵니다.
2. 매물 목록에서 원하는 매물을 클릭하여 상세 정보를 확인합니다.
3. 다음 기능들을 활용하여 매물을 찾아보세요:
   - 전세/월세/매매 거래유형 필터
   - 가격순 정렬
   - 면적순 정렬
   - 엑셀 다운로드
   - 매물 저장 및 메모 기능

## 주의사항

- 처음 실행 시 Windows 보안 경고는 정상입니다.
- 프로그램이 있는 폴더에 쓰기 권한이 필요합니다.
- 인터넷 연결이 필요합니다.

## 문제 해결

1. 프로그램이 실행되지 않는 경우:
   - 압축을 완전히 해제했는지 확인
   - 폴더에 쓰기 권한이 있는지 확인
   - 바이러스 백신 프로그램이 실행을 차단하는지 확인

2. 데이터가 표시되지 않는 경우:
   - 인터넷 연결 확인
   - "데이터 업데이트" 버튼 클릭
